HTML Table Reference

Table Tags

Element Name Function
table Table element The entire body of the table, holds all of it.
thead Table head The header of the table, i.e. the first row with the titles.
tbody Table body The body of the table, holds the bulk of information.
tfoot Table footer The end of the table, usually holds the end amount like the total of data.
tr Table row Makes the entire row, holds all the data.
td Table data Holds the data of the table, each td element will display the data of one column in the row.
TIP Every td is a new cell, so for more cells, have more td's.

Table Attributes

Element Name Function
colspan Column span Will indicate how much column space should the data take up.
rowspan Row span Will indicate how much row space should the data take up.
